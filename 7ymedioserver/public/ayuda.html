¿Que pasa si la banca se queda sin puntos al repartir puntos?

La banca repartirá en orden de prioridad.
Si la banca se queda sin puntos dejará de repartir en ese momento.
Por ejemplo, si la banca tiene 5 puntos y el primero en recibir hizo una apuesta también a 5, recibirá todos los puntos mientras que todos los que vayan después no recibirán nada.


¿Porqué recibo 0.5 punto si me ha tocado una carta de mayor valor?

Lo máximo que se puede ganar de una carta son 7 puntos.
Por lo tanto, cualquier carta que tenga un valor mayor a 7 dará 0.5 punto en vez de su valor real.

¿Que es el nivel de riesgo?

El nivel de riesgo, por decirlo de una forma, es lo que define la apuesta y la probabilidad que tiene un bot o un jugador que juegue como tal de robar una carta.
Cuanto mayor sea el nivel de riesgo mayor será la apuesta y la probabilidad de robar carta otra vez.

¿Se requiere de algún tipo de intalación a parte para poder jugar?

Si. Tal y como indicamos en el video del apartado "Tutorial" para poder ejecutar el juego correctamente necesitas 2 extensiones a parte del Python.
Necesitarás "pyfiglet" (para que los textos se generen correctamente) y la de "mysql.connector" (para poder conectarse a la base de datos que utiliza el juego).

¿Como se instala "pyfiglet" y "mysql.connector"?

Primero de todo se ha de abrir una terminal (cmd).
Para instalar "pyfiglet" has de ejecutar el siguiente comando sin comillas (" "): "sudo apt-get install python-pyfiglet" (Linux) o "pip install pyfiglet" (Windows)
Por parte de "mysql.connector" habrás de ejecutar: "pip3 install mysql-connector" (Linux) o "pip install mysql-connector-python" (Windows)
